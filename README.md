## 📖 Overview
This product, called Ghost Hunter, is a zero-config CLI tool that solves the problem of detecting 'ghost' dependencies by comparing import statements in source files against package manifests. It is designed for Python and JavaScript/TypeScript projects and can identify discrepancies between declared dependencies and actual usage. By using this tool, users can eliminate ghost dependencies and slash deployment bloat, saving time and resources. The tool is lightweight, single-file, and MIT-licensed, making it accessible to developers who want to simplify their dependency audits. Ghost Hunter is for developers who want to streamline their development process and reduce unnecessary dependencies.
